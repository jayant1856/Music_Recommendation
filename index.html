About the Project

Problem Statement

Users often struggle to find songs that match their current mood. This system uses unsupervised Machine Learning to cluster songs by audio features and recommend them by mood category.

How It Works

  • Spotify audio features extracted per track
  • K-Means clustering groups similar songs
  • Clusters mapped to mood categories
  • Top 10 results ranked by popularity
